# Changelog

## 1.0.0

- Encrypted and identity-pinned Android ↔ Windows connection.
- Automatic reconnect across LAN, hotspot, USB tethering and Bluetooth fallback.
- Bidirectional clipboard and text sharing with loop prevention.
- Streaming file transfer with SHA-256 verification and safe partial-file handling.
- Android media catalog and Windows phone-files gallery.
- Android notification forwarding with Windows filtering and privacy controls.
- DeviceSync Android keyboard with clipboard tools, local RU/EN translation, emoji and T9 V2.
- Privacy-safe diagnostics and user-confirmed support bundles.
